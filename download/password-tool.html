Offline Mode - Password Storage

✅ Good News! In offline mode, passwords are stored as plain text. You just need to find your user data in SharedPreferences and update the password field directly.

How to Change Your Password:

  1. On Android, go to: Settings → Apps → TaskMaster Pro → Storage
  2. Or use a file explorer to find SharedPreferences
  3. Look for the app's shared_prefs folder
  4. Find the stored JSON data
  5. Change the "password" field to your new password
  6. Save and restart the app

Hash Algorithm Used:

// SHA-256 Hash Function (used by backend)
async function hashPassword(password) {
  const encoder = new TextEncoder();
  const data = encoder.encode(password);
  const hashBuffer = await crypto.subtle.digest('SHA-256', data);
  const hashArray = Array.from(new Uint8Array(hashBuffer));
  return hashArray.map(b => b.toString(16).padStart(2, '0')).join('');
}

❓ FAQ

Q: Where is my data stored?

In offline mode, all data is stored in Android's SharedPreferences. The file is typically located at:
/data/data/com.taskmaster.taskmaster_pro/shared_prefs/FlutterSharedPreferences.xml

Q: Can I reset my password by clearing app data?

Yes! Clearing app data will remove all stored data including your account. You can then register again with a new password. Go to: Settings → Apps → TaskMaster Pro → Storage → Clear Data

Q: What if I'm using the backend API?

If you switch to the backend API mode, passwords are hashed with SHA-256. Use the "Backend Mode" tab above to generate the hash, then update the password field in the D1 database.
